Three new concepts

  1. HTML Attributes
    1. Attributes are always specified in the start tag
    2. All HTML elements can have attributes
    3. Attributes provide additional information about elements
  2. The Cascading Stylesheets
    1. CSS puts the designer in the driver's seat
    2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media
    3. CSS saves a lot of work. It can control the layout of multiple web pages all at once
  3. HTML Document, DOM
    1. The objects that comprise the structure and content of a document on the web.
    2. The document object is the root node of the HTML document.
